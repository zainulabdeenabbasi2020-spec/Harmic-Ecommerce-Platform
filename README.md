# Harmic - Grocery eCommerce Platform 🍏

A full-stack grocery eCommerce system featuring product management, user authentication, and order tracking.

## 🛠 Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** Oracle Database

## 🚀 Key Features
- Database-driven product catalog.
- User and Order management modules.
- RESTful API integration.
